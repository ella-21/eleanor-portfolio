# eleanor-portfolio

This repository now contains a static portfolio website that can be published with GitHub Pages.

## Publish Instructions

1. Commit and push these files to the `main` branch.
2. Go to the repository Settings > Pages.
3. Select the `main` branch and root folder (`/`).
4. Save and wait a few minutes for the site to become available.

Once published, the site should be live at:

`https://ella-21.github.io/eleanor-portfolio/`
